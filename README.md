# 🚀 Simple Login Form

A clean and responsive login form built using **HTML, CSS, and JavaScript** with real-time validation and user-friendly feedback.

---

## ✨ Features

* 🔍 **Real-time validation**
* 📧 Email format validation using Regex
* 🔒 Password minimum length check (6+ characters)
* ⚠️ Error messages shown only after user interaction
* 🚫 Submit button disabled until form is valid
* 🧮 Live character counter (Textarea)
* ⚡ Input limit with warning message

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla)** – Logic & validation

---

---

## ⚙️ How It Works

1. User enters input fields
2. Validation runs in real-time
3. Errors appear only after interaction
4. Submit button activates only when all inputs are valid
5. On submission → success alert is triggered
